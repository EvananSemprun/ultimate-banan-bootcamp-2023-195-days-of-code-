Insertar datos en una tabla en PostgreSQL

Pasos:

  1. Conexión a la base de datos:
    2.     
      <!-- Aquí va el código para conectarse a la base de datos de PostgreSQL -->
  2. Selección de la base de datos:
    3.     
      \c nombre_base_de_datos
  3. Creación de la tabla:
    4.     
      CREATE TABLE nombre_tabla (
          columna1 tipo1,
          columna2 tipo2,
          ...
      );
  4. Insertar datos:

    5. a. Insertar una fila a la vez:

        
      INSERT INTO nombre_tabla (columna1, columna2, ...)
      VALUES (valor1, valor2, ...);

    b. Insertar múltiples filas a la vez:

        
      INSERT INTO nombre_tabla (columna1, columna2, ...)
      VALUES
          (valor1_1, valor2_1, ...),
          (valor1_2, valor2_2, ...),
          ...
      ;
  5. Verificar los datos insertados:
    6.     
      SELECT * FROM nombre_tabla;
