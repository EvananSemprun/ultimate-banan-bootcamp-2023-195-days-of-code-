Nociones básicas de PostgreSQL

¿Qué es PostgreSQL?

PostgreSQL es un sistema de gestión de bases de datos relacional de código abierto y robusto.

Conceptos básicos

  • Bases de datos: Una base de datos en PostgreSQL es un contenedor para almacenar datos relacionados.
  • Tablas: Una tabla es una estructura que almacena los datos en filas y columnas.
  • Columnas: Una columna representa un atributo específico en una tabla, como el nombre o la edad.
  • Claves primarias: Una clave primaria es una columna o un conjunto de columnas que identifican de manera única cada fila en una tabla.
  • Relaciones: Las relaciones establecen vínculos entre tablas mediante el uso de claves primarias y foráneas.

Ejemplo de creación de una tabla en PostgreSQL

        
            CREATE TABLE usuarios (
                id SERIAL PRIMARY KEY,
                nombre VARCHAR(50),
                edad INTEGER
            );

Ejemplo de consulta SELECT en PostgreSQL

        
            SELECT * FROM usuarios;

Ejemplo de inserción de datos en PostgreSQL

        
            INSERT INTO usuarios (nombre, edad)
            VALUES ('Juan', 25);

Ejemplo de actualización de datos en PostgreSQL

        
            UPDATE usuarios
            SET edad = 26
            WHERE nombre = 'Juan';

Ejemplo de eliminación de datos en PostgreSQL

        
            DELETE FROM usuarios
            WHERE nombre = 'Juan';
